# 향수 추천 ChatGPT

## 1. 목표와 기능
### 1.1 목표
- 남성과 여성을 위한 데일리 향수 추천 AI
### 1.2 기능
- user가 조건을 작성 후 추천받기 버튼을 누르면 ChatGPT API를 통해 향수를 추천해줍니다.

### 2.1 개발 환경
- HTML, CSS, JavaScript
